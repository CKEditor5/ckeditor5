Error codes

CKEditor 5 Framework logs errors and warnings to the console. The following list contains more detailed descriptions of those issues.

  • attribute-operation-attribute-exists

    The attribute with given key already exists for the given node.

    Parameters

    node : Node
    key : String

  • batch-merge-no-element-after

    Node after merge position must be an element.

  • batch-merge-no-element-before

    Node before merge position must be an element.

  • batch-move-range-not-flat

    Range to move is not flat.

  • batch-register-taken

    This batch method name is already taken.

    Parameters

    name : String

  • batch-removeMarker-no-marker

    Trying to remove marker that does not exist.

  • batch-rename-not-element-instance

    Trying to rename an object which is not an instance of Element.

  • batch-setMarker-no-range

    Range parameter is required when adding a new marker.

  • batch-split-root

    Root element can not be split.

  • batch-unwrap-element-no-parent

    Trying to unwrap an element which has no parent.

  • batch-wrap-element-attached

    Element to wrap with is already attached to a tree model.

  • batch-wrap-element-not-empty

    Element to wrap with is not empty.

  • batch-wrap-range-not-flat

    Range to wrap is not flat.

  • build-model-converter-element-to-attribute

    To-attribute conversion is supported only for model attributes.

  • build-model-converter-non-marker-to-highlight

    To highlight conversion is supported only for model markers.

  • ckfinder-upload-adapter-no-config

    Please provide ckfinder.uploadUrl config option.

  • collection-add-invalid-id

    This item's id should be a string.

  • collection-add-item-already-exists

    This item already exists in the collection.

  • collection-add-item-bad-index

    The index number has invalid value.

  • collection-bind-to-rebind

    The collection cannot be bound more than once.

  • collection-get-invalid-arg

    Index or id must be given.

  • collection-remove-404

    Item not found.

  • commandcollection-command-not-found

    Command does not exist.

    Parameters

    commandName : String

    Name of the command.

  • componentfactory-item-exists

    The item already exists in the component factory.

    Parameters

    name : String

    The name of the component.

  • componentfactory-item-missing

    There is no such UI component in the factory.

    Parameters

    name : String

    The name of the missing component.

  • contextualballoon-add-view-exist

    Trying to add configuration of the same view more than once.

  • contextualballoon-remove-view-not-exist

    Trying to remove configuration of the view not defined in the stack.

  • delta-fromjson-no-deserializer

    This delta has no defined deserializer.

    Parameters

    name : String

  • document-applyOperation-wrong-version

    Only operations with matching versions can be applied.

    Parameters

    operation : Operation

  • document-createRoot-name-exists

    Root with specified name already exists.

    Parameters

    doc : Document
    name : String

  • document-getRoot-root-not-exist

    Root with specified name does not exist.

    Parameters

    name : String

  • document-selection-wrong-position

    Range from document selection starts or ends at incorrect position.

    Parameters

    range : Range

  • documentselection-cannot-create

    DocumentSelection#createFromSelection() is not available. There can be only one DocumentSelection per document instance, so creating new DocumentSelections this way would be unsafe.

  • history-wrong-version

    Given base version points to the middle of a delta.

  • liveposition-root-not-rootelement

    LivePosition root has to be an instance of RootElement.

  • marker-destroyed

    Operating on destroyed marker instance.

  • marker-destroyed

    Operating on destroyed marker instance.

  • marker-destroyed

    Operating on destroyed marker instance.

  • model-schema-item-exists

    Item with specified name already exists in schema.

  • model-schema-no-item

    Item with specified name does not exist in schema.

  • model-selection-added-not-range

    Trying to add an object that is not an instance of Range.

  • model-selection-moveFocusTo-no-ranges

    Cannot set selection focus if there are no ranges in selection.

  • model-textproxy-wrong-length

    Given length value is incorrect.

  • model-textproxy-wrong-offsetintext

    Given offsetInText value is incorrect.

  • model-writer-move-range-not-flat

    Trying to move a range that starts and ends in different element.

  • model-writer-remove-range-not-flat

    Trying to remove a range that starts and ends in different element.

  • move-operation-node-into-itself

    Trying to move a range of nodes into one of nodes from that range.

  • move-operation-nodes-do-not-exist

    The nodes which should be moved do not exist.

  • move-operation-position-invalid

    Source position or target position is invalid.

  • move-operation-range-into-itself

    Trying to move a range of nodes into the middle of that range.

  • node-not-found-in-parent

    The node's parent does not contain this node.

  • node-not-found-in-parent

    The node's parent does not contain this node.

  • nodelist-index-out-of-bounds

    Given index cannot be found in the node list.

  • nodelist-insertNodes-not-node

    Trying to insert an object which is not a Node instance.

  • nodelist-offset-out-of-bounds

    Given offset cannot be found in the node list.

  • observable-bind-duplicate-attrs

    Attributes must be unique.

  • observable-bind-no-callback

    Binding multiple observables only possible with callback.

  • observable-bind-rebind

    Cannot bind the same attribute more that once.

  • observable-bind-to-attrs-length

    The number of attributes must match.

  • observable-bind-to-extra-callback

    Cannot bind multiple attributes and use a callback in one binding.

  • observable-bind-to-parse-error

    Invalid argument syntax in to().

  • observable-bind-wrong-attrs

    All attributes must be strings.

  • observable-set-cannot-override

    Cannot override an existing property.

    This error is thrown when trying to set an attribute with a name of an already existing property. For example:

    let observable = new Model();
observable.property = 1;
observable.set( 'property', 2 );        // throws

observable.set( 'attr', 1 );
observable.set( 'attr', 2 );            // ok, because this is an existing attribute.

  • observable-unbind-wrong-attrs

    Attributes must be strings.

  • observablemixin-cannot-decorate-undefined

    Cannot decorate an undefined method.

    Parameters

    object : Object

    The object which method should be decorated.

    methodName : String

    Name of the method which does not exist.

  • operation-attribute-wrong-old-value

    Changed node has different attribute value than operation's old attribute value.

    Parameters

    item : Item
    key : String
    value : *

  • plugincollection-load

    It was not possible to load the plugin.

    Parameters

    plugin : String

    The name of the plugin that could not be loaded.

  • plugincollection-plugin-name-conflict

    Two plugins with the same pluginName were loaded. This may lead to runtime conflicts between these plugins. This usually means that incorrect params were passed to create.

    Parameters

    pluginName : String

    The duplicated plugin name.

    plugin1 : function

    The first plugin constructor.

    plugin2 : function

    The second plugin constructor.

  • plugincollection-plugin-not-found

    Some plugins are not available and could not be loaded.

    Plugin classes (constructors) need to be provided to the editor before they can be loaded by name. This is usually done by the builder by setting the build property.

    Parameters

    plugins : Array.<String>

    The name of the plugins which could not be loaded.

  • plugincollection-required

    Cannot load a plugin because one of its dependencies is listed in the removePlugins option.

    Parameters

    plugin : function

    The required plugin.

    requiredBy : function

    The parent plugin.

  • position-after-root

    You can not make position after root.

    Parameters

    root : Item

  • position-after-root

    You can not make a position after a root.

    Parameters

    root : Node

  • position-before-root

    You can not make position before root.

    Parameters

    root : Item

  • position-before-root

    You cannot make a position before a root.

    Parameters

    root : Node

  • position-fromjson-no-root

    Cannot create position for document. Root with specified name does not exist.

    Parameters

    rootName : String

  • position-parent-incorrect

    Position parent have to be a model element or model document fragment.

  • position-path-incorrect

    Position path must be an Array with at least one item.

    Parameters

    path :

  • position-root-invalid.

    Position root invalid.

  • range-create-from-ranges-empty-array

    At least one range has to be passed.

  • rect-source-not-in-dom

    The Rect class depends on getBoundingClientRect and getClientRects DOM methods. If the source of a rect in an HTML element or a DOM range but it does not belong to any rendered DOM tree, these methods will fail to obtain the geometry and the rect instance makes little sense to the features using it.

    To get rid of this warning make sure the source passed to the constructor is a descendant of window.document.body.

    Parameters

    source : String

    The source of the Rect instance.

  • rename-operation-wrong-name

    Element to change has different name than operation's old name.

  • rename-operation-wrong-position

    Given position is invalid or node after it is not instance of Element.

  • renderer-unknown-type

    Unknown type passed to Renderer.markToSync.

  • rootattribute-operation-attribute-exists

    The attribute with given key already exists for the given node.

    Parameters

    root : RootElement
    key : String

  • rootattribute-operation-wrong-old-value

    The attribute which should be removed does not exists for the given node.

    Parameters

    root : RootElement
    key : String
    value : *

  • rootattributeoperation-fromjson-no-root

    Cannot create RootAttributeOperation for document. Root with specified name does not exist.

    Parameters

    rootName : String

  • selection-range-intersects

    Trying to add a range that intersects with another range from selection.

    Parameters

    addedRange : Range

    Range that was added to the selection.

    intersectingRange : Range

    Range from selection that intersects with addedRange.

  • selection-range-intersects

    Trying to add a range that intersects with another range from selection.

    Parameters

    addedRange : Range

    Range that was added to the selection.

    intersectingRange : Range

    Range from selection that intersects with addedRange.

  • selectionchange-infinite-loop

    Selection change observer detected an infinite rendering loop. Most probably you try to put the selection in the position which is not allowed by the browser and browser fixes it automatically what causes selectionchange event on which a loopback through a model tries to re-render the wrong selection and again.

  • template-extend-render

    Extending a template after rendering may not work as expected. To make sure the extending works for the rendered element, perform it before render is called.

  • toolbarview-item-unavailable

    There was a problem processing the configuration of the toolbar. The item with the given name does not exist so it was omitted when rendering the toolbar.

    This warning usually shows up when the Plugin which is supposed to provide a toolbar item has not been loaded or there is a typo in the configuration.

    Make sure the plugin responsible for this toolbar item is loaded and the toolbar configuration is correct, e.g. Bold is loaded for the 'bold' toolbar item.

    Parameters

    name : String

    The name of the component.

    Neither boundaries nor starting position have been defined.

  • tree-walker-no-start-position

    Neither boundaries nor starting position have been defined.

  • ui-viewcollection-init-reinit

    Item may be converted to either model node or model document fragment.

    Cannot add children to EmptyElement.

    Before focusing view document, selection should be placed inside one of the view's editables. Normally its selection will be converted from model document (which have default selection), but when using view document on its own, we need to manually place selection before focusing it.

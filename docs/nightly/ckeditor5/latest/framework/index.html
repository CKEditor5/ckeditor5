CKEditor 5 Framework

CKEditor 5 Framework is a set of components enabling you to create any kind of text editing solution.

Use the navigation tree on the left to navigate through CKEditor 5 Framework documentation.

Related links

  • Features – Learn about some of the features included in the CKEditor 5 Framework.

Framework documentation is still in progress, so it may be still lacking many things. Feel free to suggest documentation enhancements and share with us your feedback about the framework.

If the documentation is insufficient, please do not be afraid of looking into the source code of CKEditor 5 packages. For example if you plan to create a new feature, check if a similar feature already exists and try to take an inspiration from its source code.
